## Requirements
- macOS 14.0+
- Xcode 15.0+

## Installation
1. Open Xcode.
2. Click on **"Clone Git Repository"**.
3. Paste the following URL: `https://github.com/DanielePerrupane/MagicTranslate.git`
4. Click **"Clone"**.
5. Build and run the project in Xcode.
